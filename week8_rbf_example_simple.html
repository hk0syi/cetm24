Radial Basis Functions and Classification

Radially Symmetric Property

  • Each Radial Basis Function (RBF) neuron has a center (c_j).
  • Its activation depends only on the distance from the input (x) to the center:

$$ \phi(x) = \phi(\|x - c_j\|) $$

  • Because of this, the response is the same in all directions around the center.
  • In 2D, the activation looks like a circular bump; in 3D, its a spherical bump; in higher dimensions, its a hypersphere.

Why This Helps Classification

  • The network doesnt care about the direction of the input relative to the center only the distance.
  • Each neuron acts like a 1D illustration of the bubble of influence around its center.
  • Inputs that fall close to a center (inside its bubble) activate that neuron strongly.

Discovering Groups (Classes)

  • During training, RBF centers are usually placed near clusters of data (e.g., using kmeans).
  • Each center + radial function effectively represents one local region of the input space.
  • The output layer then learns to combine these radial activations to decide which class the input belongs to.

Key Idea:
The radially symmetric property is exactly what allows RBFs to discover which group (class) an input belongs to.
Inputs close to a classs centers high activation classified into that group.

# -----------------------------
# Step 4: Define RBF Function
# -----------------------------
rbf <- function(x, center, d_max, m1) {
  dist2 <- sum((x - center)^2)
  phi <- exp(-(m1 / (d_max^2)) * dist2)
  return(phi)
}

# -----------------------------
# Step 7: Define Prediction Function (forward pass)
# -----------------------------
rbf_predict <- function(newX, centers, d_max, m1, W) {
  H_new <- matrix(0, nrow=nrow(newX), ncol=nrow(centers))
  for (i in 1:nrow(newX)) {
    for (j in 1:nrow(centers)) {
      H_new[i,j] <- rbf(newX[i,], centers[j,], d_max, m1)
    }
  }
  y_pred <- H_new %*% W
  class_pred <- apply(y_pred, 1, which.max)
  return(list(predictions=y_pred, class=class_pred))
}

# ================================================================
# RBF Neurons: Global  vs Local  (1D Visualization)
# ================================================================

# -----------------------------
# Step 1: Define centers
# -----------------------------
centers <- c(2, 5, 8)  # 3 neuron centers
k <- length(centers)

# -----------------------------
# Step 2: Compute Global 
# -----------------------------
d_max <- max(centers) - min(centers)
sigma_global <- d_max / sqrt(k)

# -----------------------------
# Step 3: Compute Local  (nearest neighbor distance / 2)
# -----------------------------
sigmas_local <- numeric(k)
for (i in 1:k) {
  others <- centers[-i]
  nearest <- min(abs(others - centers[i]))
  sigmas_local[i] <- nearest/2
}

cat("Global  =", sigma_global, "\n")
cat("Local  per neuron =", sigmas_local, "\n")

# -----------------------------
# Step 4: Define RBF function
# -----------------------------
rbf <- function(x, center, sigma) {
  exp(- (x-center)^2 / (2*sigma^2))
}

# -----------------------------
# Step 5: Generate curve data
# -----------------------------
x_vals <- seq(0, 10, length.out=200)

# Global  curves
H_global <- sapply(1:k, function(j) rbf(x_vals, centers[j], sigma_global))

# Local  curves
H_local <- sapply(1:k, function(j) rbf(x_vals, centers[j], sigmas_local[j]))

# -----------------------------
# Step 6: Plot Global 
# -----------------------------
par(mfrow=c(1,2))  # two plots side by side
matplot(x_vals, H_global, type="l", lwd=2, lty=1,
        col=c("red","blue","green"),
        main="Global  (uniform width)",
        xlab="x", ylab="Activation")
points(centers, rep(0,k), pch=19, col=c("red","blue","green"))

# -----------------------------
# Step 7: Plot Local 
# -----------------------------
matplot(x_vals, H_local, type="l", lwd=2, lty=1,
        col=c("red","blue","green"),
        main="Local  (per neuron width)",
        xlab="x", ylab="Activation")
points(centers, rep(0,k), pch=19, col=c("red","blue","green"))

par(mfrow=c(1,1))  # reset layout
