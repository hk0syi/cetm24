Understanding the Scales of Measurement

In statistics, there are four main scales of measurement:

1. Nominal

Used to label variables in different classifications and does not imply a quantitative value or order. For example, Height can be classified as "above standard height" or "below standard height."

2. Ordinal

Represents non-mathematical ideas such as frequency, satisfaction, or ranking. For example, Height can be "short," "middle," or "tall."

3. Interval

A numerical scale where the order and difference between variables is known, but there is no true zero. For example, Score ranges (0-49, 50-60, etc.) are interval data.

4. Ratio

A variable measurement scale that provides order, difference between values, and a true zero. For example, continuous height measurements like 152cm or 169cm.
